# 💱 Currency Converter

A responsive Currency Converter web application built using HTML, CSS, and JavaScript. It fetches real-time exchange rates using an exchange rate API.

## 🚀 Live Demo

https://currency-converter-raut.vercel.app

## ✨ Features

- Convert currencies in real time
- Supports multiple currencies
- Responsive design
- Country flags
- Clean and simple UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Exchange Rate API
- Vercel (Deployment)
